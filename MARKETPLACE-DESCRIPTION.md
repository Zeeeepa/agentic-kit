# Agentic Toolkit - Marketplace Listing

## Plugin Name & Tagline
**Agentic Toolkit** - AI-Powered Software Development & Team Orchestration Platform

## Overview
Agentic Toolkit is a comprehensive AI-driven software development platform featuring 13 specialized agents and up to 16 powerful skills. It orchestrates complete workflows from product strategy through development, design, testing, and deployment. Perfect for teams looking to accelerate their development cycle with intelligent AI agents managing distinct roles and responsibilities.

## Key Features

### Role-Based AI Agents (13 Total)
- **Product Strategy**: Master, Product Manager, Product Owner for strategic direction
- **Technical Delivery**: Full-Stack Developer, Holistic Architect for implementation and design
- **Quality & Testing**: QA Test Architect for comprehensive testing strategies
- **Team Leadership**: Scrum Master for project management and orchestration
- **User Experience**: UX Expert for interface and experience design
- **Business Analysis**: Business Analyst for requirements and analysis
- **Process Automation**: Orchestrator for workflow management
- **Task Processing**: Create PRD, Generate Tasks, Process Task Lists for structured work

### Specialized Skills (Variant-Dependent)
- **Document Processing**: PDF manipulation, Word document creation (DOCX), spreadsheet analysis (XLSX)
- **Presentation & Design**: PowerPoint creation (PPTX), visual design with Canvas, algorithmic art generation
- **Theme & Branding**: Theme Factory for consistent styling, Brand Guidelines compliance
- **Development Tools**: MCP Server builder, Custom skill creator, Web app testing toolkit
- **Communication**: Internal communications templates and resources
- **Advanced Features** (Pro only): Artifacts builder for complex HTML, Slack GIF creation

## Installation & Quick Start

### Install from Claude Code Marketplace
```bash
# Standard variant (recommended)
claude install agentic-kit

# Lite variant for minimal setups
claude install agentic-kit-lite

# Pro variant for full capabilities
claude install agentic-kit-pro
```

### Initial Setup
1. After installation, all 13 agents become available in Claude Code
2. Use the Master agent to orchestrate workflows
3. Invoke specific agents as needed (Product Manager, Developer, etc.)
4. Access skills directly from agent contexts
5. Leverage resource templates for common workflows

## Use Cases

### Product Development
Create comprehensive PRDs with the Product Manager, generate tasks, manage backlogs with Product Owner, architect systems with the Holistic Architect, and execute with the Full-Stack Developer.

Use QA Test Architect to design testing strategies, coordinate with the Scrum Master for sprint planning, and ensure comprehensive coverage across all components.

### Team Coordination
The Orchestrator and Scrum Master manage complex multi-agent workflows, coordinate team efforts, and maintain project momentum through structured task management.
